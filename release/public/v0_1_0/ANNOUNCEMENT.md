# Azimuth Bench — v0.1.0 release candidate (announcement draft)

**Azimuth Bench** is an open-source **inference benchmark** toolchain: run a fixed throughput suite against **MLX**, **OpenAI-compatible HTTP**, or **Ollama** backends, emit JSON artifacts, and build **static** HTML + JSON reports with explicit **comparability** rules—no hosted app required.

This **v0.1.0** tag is a **first public release candidate**: documentation and packaging hygiene, not a claim of ecosystem maturity. The repo includes a **reference** `benchmarks/` tree you can turn into reports **without rerunning** benchmarks, plus tests and a clear source-of-truth boundary.

**Docs:** see the repository `README.md`, `CONTRIBUTING.md`, `SECURITY.md`, and `docs/azimuth_bench/SOURCE_OF_TRUTH.md`.

**Not in scope for v0.1.0:** llama.cpp/vLLM production adapters, hosted UI, PyPI automation, or benchmark universality claims.
